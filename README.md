# Claude Code Plugin Marketplace

Plugin Marketplace by **SoftwareEngineering Hauschel** — [hauschel.de](https://hauschel.de)

## Available MCP Servers

| MCP Server | Description |
|------------|-------------|
| [jdt-mcp-server](https://github.com/hauschel-ai-tools/jdt-mcp-server) | Eclipse Plugin embedding an MCP Server to expose JDT features to AI coding assistants like Claude Code and Cursor |

## Installation

```bash
# Add marketplace
/plugin marketplace add https://github.com/hauschel-ai-tools/claude-code-marketplace.git

# Install MCP server
/plugin install jdt-mcp-server@hauschel-plugins
```
